# Iris Flower Classification using Decision Trees

A clean, beginner-friendly Machine Learning project that trains a **Decision Tree Classifier** on the classic **Iris dataset** to predict flower species with high accuracy.

## 📌 Project Overview
This repository contains a simple Python script using [scikit-learn](https://scikit-learn.org) to classify Iris flowers into three species (*Setosa*, *Versicolor*, and *Virginica*) based on four features: sepal length, sepal width, petal length, and petal width. 

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.x installed along with the required libraries. 

### Installation
1. Clone this repository:
   ```bash
   https://github.com/oishi-sarker/Task-2-OishiSarker
   ```

2. Install the necessary dependencies:
   ```bash
   pip install scikit-learn
   ```

### Running the Code
Execute the script using Python:
```bash
python main.py
```
